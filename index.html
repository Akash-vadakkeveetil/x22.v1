The Mediport

2024

  • Spring Boot
  • Java
  • MySql
  • Web Dev

The MediPort is a web-based pharmacy and hospital inventory system that streamlines the tracking, management, and ordering of medicines. It replaces manual record-keeping with a centralized platform that monitors stock levels, expiry dates, and supplier details in real time. The system automatically updates inventory after each transaction, sends alerts for low stock, and facilitates direct communication with suppliers for efficient restocking. With role-based access, MediPort offers secure, customized interfaces for administrators, pharmacists, and suppliers, while generating detailed reports to support data-driven decisions.

Heisei

2025

  • Node Js
  • Express Js
  • Blockchain
  • MySql
  • Web Dev

Heisei is an advanced inventory management system powered by blockchain technology to ensure secure, transparent, and tamper-proof transaction records. Acting as a middle layer between the frontend/backend and the database, it uses a custom-built Heisei Chain in Java for transaction verification, while inventory data is stored off-chain for efficiency.

Key Features:

Blockchain Integration: Guarantees immutability of all inventory records.
Transaction Verification: Each update is validated on-chain to prevent unauthorized changes.
Role-Based Access: Secure login system for donors and camps.
Decentralized Trust: Eliminates tampering through a transparent ledger.
Efficient Tracking: Enables real-time monitoring and reliable supply management.

Heisei combines blockchain security with streamlined inventory control—enhancing transparency, trust, and operational efficiency.

Heisei-Chain

2025

  • Java Springboot
  • Core Java
  • Sha-256
  • Web Dev

Heisei Chain is a lightweight, private blockchain built with Java and Spring Boot to ensure secure, immutable, and verifiable transaction records for inventory management. It serves as the core of the "Heisei" system, making all stock updates tamper-proof and auditable.

Key Features:

Custom Blockchain: Built from scratch in Java with SHA-256 hashing for data integrity.
Immutable Ledger: Cryptographically linked transactions prevent unauthorized changes.
Spring Boot Integration: Enables seamless API communication with the inventory system.
Off-Chain Storage: Transactions verified on-chain; inventory data stored in MySQL for efficiency.
Consensus Mechanism: Kafka-based model ensures consistency across distributed environments.
Real-Time Verification: Each transaction is validated before being added to the chain.

Heisei Chain strengthens supply chain transparency with a secure, decentralized, and tamper-resistant ledger.
